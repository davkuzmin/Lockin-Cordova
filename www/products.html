Vault

Vault is a secure and beautiful cloud storage product. It provides state of th art security protocols along with design that is unmatched. We offer consumer and business storage plans to fit your needs and "pay for what you use" pricing available for consumers.

Security

Security is very important for us here at Lockin. We did research on the industry standards and decided to take it a huge leap forward to make sure that the users get the most up-to-date security protocols. We will constantly be working on the security of your data no matter what the cost or inconvience. Vault has been tested over and over to make sure that hackers stay out, forever.

Design

We worked for months on the design of Vault. We wanted it to be easy to use and enable everyone to be able to enjoy using it. Finally we mixed bootstrap and our own modern look and found a common ground that our users love.

Business

Vault is not only for consumers. Businesses can use our enterprise version of Vault to store their important files. With administrative features and larger storage sizes, it has never been easier for a business to secure their data in the cloud.

Speed Results

When there are a multitude of online file storage options, why choose Lockin? In our speed tests, we weren’t as fast as the bigger names in file storage, but we are only behind in most cases, a few seconds or less. Now some would argue that speed is the most important when a user is interacting with an online service, and it is an important factor when we are making and implementing features in Vault, but what we lack in speed we make up for in security. Sacrificing a few seconds or less, our algorithms and security protocols ensure that everything you put into Vault is protected. We start by encrypting your basic data like your username and password using 2048/3072 bit encryption which is currently above the industry standard. Then with every file you upload we use a HTTPS/SSL secure file stream to transport your files. Lastly, your files are stored using AES 256 bit encryption, the same method as the big name file storage services. So we offer a product that is barely slower than established competition, but with above industry security standards. But just because we have better standards doesn’t mean we are stopping there. Future updates will improve our security even higher above the standard for online file storage. A little less speed, for a lot more security, we believe thats a fair trade off.
